---

## Features

- Animated nav logo — 8 rotating data-analyst modes
- Hero headline with flipping word (signal / pattern / truth / proof / insight...)
- Scroll-driven stacked project cards — 3 data science projects
- Orbital skills carousel — 6 cards, scroll-driven rotation, hover to flip
- About section with word-by-word animated quote
- LinkedIn-linked avatar (hover to reveal)
- Power BI manufacturing dashboard cross-link
- Closing section with flipping word
- Apple-style slide-to-email with confetti burst at 50%
- Fully responsive
